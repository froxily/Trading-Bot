# Trading Simulator

A real-time stock trading simulator built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Live market simulation with 10 different stocks
- Real-time price charts using Recharts
- Portfolio management with buy/sell functionality
- News events that impact stock prices
- Leaderboard to track top traders
- Order history tracking
- LocalStorage persistence

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will open at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Recharts (for charts)
- Lucide React (for icons)

## How to Play

1. Click "Set Name" to enter your trader name
2. Select a stock from the market list
3. Enter the number of shares you want to trade
4. Click BUY or SELL to execute trades
5. Watch the live charts and news events
6. Try to maximize your portfolio value!
